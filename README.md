# 📝 To-Do List App (C# WinForms)

A lightweight desktop To-Do List application created using **C# and .NET Windows Forms**.
This project was built as a practical learning exercise focused on desktop UI development, event handling, and local data persistence.

---

## ✨ Features

* Add new tasks
* Mark tasks as completed
* Delete selected tasks
* Automatic task saving
* Completed tasks removed after application shutdown
* Local settings folder storage

---

## 🛠 Technologies Used

* C#
* .NET Windows Forms
* File I/O
* Event-driven programming

---

## 📂 Data Storage

Tasks are saved locally inside:

Settings/tasks.txt

The application automatically creates the folder if it does not exist.

---

## ▶ How to Run

1. Clone the repository
2. Open the solution in Visual Studio
3. Build and run the project
